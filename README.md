# BuggyTasks — тестовое приложение для manual QA

Небольшое одностраничное веб‑приложение (3 экрана) с намеренно заложенными простыми багами. Подходит, чтобы показывать на собеседованиях для ручного тестирования: кандидат может исследовать интерфейс и искать дефекты.

## Как запустить

- **Шаг 1**: откройте файл `index.html` в любом браузере (двойной клик по файлу или "Open With" → браузер).
- Никакого бэкенда и сборки не требуется, всё работает целиком в браузере.

## Структура экранов

- **Экран "Вход"**: простая форма логина с e‑mail, паролем и ссылкой "Забыли пароль?".
- **Экран "Задачи"**: список задач с добавлением, фильтрами, поиском и мини‑статистикой (фильтр по приоритету).
- **Экран "Профиль"**: информация о пользователе, настройки уведомлений и тумблер тёмной темы.

## Идея для использования на собеседовании

- Попросить кандидата:
  - описать, какие сценарии он бы тестировал на каждом из экранов;
  - последовательно проходить сценарии и фиксировать найденные баги;
  - при желании — составить тест‑кейсы и баг‑репорты (с шагами, фактическим/ожидаемым результатом и приоритетом).
